# Average People

## Usage

```bash
npm install
npm run person
```

example output:
```json
{
    lon: 24.028331173076737,
    lat: -10.91891826035912,
    country_code: 'zm',
    'ISO3166-2-lvl4': 'ZM-06'
}
```

## Data Used

- Geospatial Population Data[1]
- OpenStreetMap reverse geocoding services[2]
